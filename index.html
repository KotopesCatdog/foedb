Справка

Добро пожаловать в Базу Зданий FoE!

  • Клик на карточке здания - показ-скрытие данных
  • Удержание ЛКМ на здании добавляет его в корзину
  • Клик по колонке в корзине — сортировка
  • Колонки можно перетаскивать
  • Выбор эпохи и параметров для фильтрации
  • Новое! Кнопка "📂 Загрузить JSON" позволяет загружать локальные JSON файлы
  • Кнопка "🌐 На сервер" возвращает загрузку данных с сервера
  • Внимание! Ведутся работы, возможны неточности
  • Для загрузки последних изменений обновляйте страницу Ctrl+F5
